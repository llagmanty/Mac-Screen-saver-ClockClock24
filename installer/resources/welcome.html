New Version

ClockClock24

Version 2.2  ·  Replaces version 2.1

ClockClock24 is a macOS screen saver inspired by the ClockClock 24 sculpture by Humans Since 1982. Twenty-four analog clocks are arranged in a 3 × 8 grid and rotate their hands to display the current time in HH:MM.

What's new in version 2.2

  • Smooth choreography. The 40-second kinetic phase now uses a single pattern for the entire duration — no more mid-choreography switches that caused visible jumps.
  • Seamless entry and exit. Hands now flow directly from the time display into choreography with zero jump. Per-clock angle offsets are computed at the transition point so motion is continuous from the very first frame.

Previously in version 2.1

  • Flat hand tips. Clean perpendicular cut at the outer ends of each hand.
  • Author signature. Rendered from SVG path data in the bottom-right corner of the wall.
  • Signed & notarised by Apple. Installs without any Gatekeeper warning on macOS 13 or later.

This installer places ClockClock24.saver in ~/Library/Screen Savers/ and replaces any previous version. Open System Settings → Screen Saver after installation to activate it.
