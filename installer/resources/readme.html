Installation & Usage

What gets installed

ClockClock24.saver is copied to ~/Library/Screen Savers/ (your user account only). No system files are modified.

Activating the screen saver

  1. Open System Settings.
  2. Go to Screen Saver.
  3. Select ClockClock24 from the list.

Uninstalling

Delete ~/Library/Screen Savers/ClockClock24.saver and choose a different screen saver in System Settings.

Compatibility

  • macOS 13 Ventura or later
  • Apple Silicon and Intel (universal binary)
